Help with AI Hallucinations

If you notice the AI is listing files or claiming data exists in directories where it doesn't actually exist, it's hallucinating. Common forms of hallucination include:

  • Describing files from a subdirectory as if they exist in a parent directory
  • Claiming to have read content from non-existent files
  • Inventing file names that don't exist in the requested directory
  • Presenting information as fact when it doesn't appear in the actual output

The structured format above is specifically designed to make the exact file locations and content clear to reduce hallucinations.
